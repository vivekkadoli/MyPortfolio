AI Image Generator

AI Image Generator is a user-friendly web application enabling users to create AI-generated images from custom text prompts. Utilizing the Hugging Face API, the app supports model selection, aspect ratio choice, and generates multiple images per prompt. Users can instantly preview and download their creations, and enjoy a modern interface that supports both dark and light modes. This project demonstrates how state-of-the-art AI can be seamlessly integrated into interactive web applications for creative experimentation and practical use, ideal for both newcomers and developers exploring image generation.

Key Features

  • AI-powered app: user text → AI-generated images (via Hugging Face API).
  • Model, aspect ratio, image selection, instant preview & download. Dark/light mode.

Workflow

  • User enters a text prompt and chooses a model and aspect ratio.
  • Client validates inputs and prepares the request payload.
  • Frontend sends the prompt and options to the Hugging Face Inference API.
  • API returns one or multiple generated images based on user selection.
  • Images render instantly in the gallery with preview capability.
  • User can download selected images; UI supports dark/light mode.
